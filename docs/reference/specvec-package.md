# specvec: Abundance-Aware Species and Community Embeddings

Learns species and community embeddings from ecological co-occurrence
data with abundance-aware weighting. Turns a long 'plot x species (x
abundance)' table into species vectors and first-class community
vectors, and ships a benchmark harness that scores embedding methods
(correspondence analysis, pointwise mutual information,
abundance-weighted PMI, and GloVe) on neutral ecological tasks. A method
is a registered '(weighting, factorization)' pair, so new methods are
added by registration rather than by copying code.
